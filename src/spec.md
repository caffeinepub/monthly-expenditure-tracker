# Specification

## Summary
**Goal:** Build a monthly expense tracker that records expenses with categories, tracks monthly earnings, and calculates savings.

**Planned changes:**
- Create backend data models for expense entries (amount, category, description, date) and monthly earnings (amount, month, year)
- Implement backend functions to add, update, delete, and retrieve expenses
- Implement backend functions to set and retrieve monthly earnings
- Implement backend calculations for category-wise expense totals and savings (earnings minus expenses)
- Create frontend form to add expense entries with amount, category, description, and date
- Create frontend form to set monthly earnings for specific months
- Create dashboard displaying category-wise expense breakdown with percentages
- Create summary section showing total earnings, total expenses, and calculated savings
- Create list view of all expense entries with edit and delete options
- Implement month/year selector to view data for different months
- Apply warm, approachable visual theme with card-based layout and clear typography

**User-visible outcome:** Users can track their monthly expenses by category, set their monthly earnings, and see a breakdown of spending with calculated savings for any selected month.
